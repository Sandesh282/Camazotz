# Phase 1 (Design Disaster)

## Objective
Our application is a club member application for Axios, the technical wing of IIIT Lucknow. 
It has a login page, and displays the information about the various different wings of Axios, and their members.


## Intentional UX Issues Introduced

### Issue 1

### Type: Navigation

### What was done:
The issue is that we cannot move from the first screen to anywhere.

### How to reproduce:
Download the apk, or open up the app in the Android Studio Emulator. When you click on the singular button, it does not function as expected and leads to nowhere???

### Why this was done:
It was done for you to struggle.

### Expected User Difficulty:
It is expected of you to not struggle at all!


### Issue 2

### Type: Semantics 

### What was done:
Quite a lot of typos are present in the current codebase, some even visible in the application itself!

### How to reproduce:
Download the apk, or open up the app in the Android Studio Emulator. You will find typos in quite a lot of places.

### Why this was done:
Due to the laziness of the developer

### Expected User Difficulty:  
This should be fairly easy..?

### Issue 3


### Type: CTA

### What was done:
When you manage to proceed past the first page, there you will find many a buttons. Do try to fix them all.

### How to reproduce:
Download the apk, or open up the app in the Android Studio Emulator.
If you manage to get to the second page somehow, then you will land at the page that is our Axios Wings page.

### Why this was done:  
This was not done on purpose.

This one should be a little bit harder than usual..

# Design Anti-Patterns Used

Mutiple CTAs have been used, where you can see them. 
Poor visual hierarchy is readily apparent. Hidden actions are also everywhere.
Navigation is very confusing, you can see it everywhere.
Many components are inconsistent with each other, sporting a variety of different colors.
